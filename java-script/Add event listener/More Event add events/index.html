KeyDown Event

keydown event occurs when a key is pressed down

element.addEventListener('keydown',funcction);


Event Object

whenever an event happens, the browser creates an Event Object it consists of information about the event that has happend

it consists of many properties and methods:
  • type
  • target
  • key
  • timeStamp
  • stopPropagation many more...

    • Keyup event

    keyup event occurs when a key is released.

    element.addEventListener('keyup',funcction);
